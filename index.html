// 02 · Fundamento matemático

Temas con ejemplos

Cada concepto explicado con su fórmula y un ejemplo práctico paso a paso.

TEMA 1 & 2

Media / Promedio (μ)

La media aritmética es la suma de todos los valores dividida entre la cantidad de datos.

μ = (x₁ + x₂ + … + xₙ) / n

📊 Ejemplo: Calificaciones de un alumno

Datos: 72, 85, 90, 78, 95, 88

μ = (72 + 85 + 90 + 78 + 95 + 88) / 6 = 508 / 6 ≈ 84.67
Suma total508
Cantidad de datos (n)6
Media84.67

✔ El promedio de 84.67 indica un rendimiento notable.

TEMA 3

Mediana

La mediana es el valor central cuando los datos están ordenados.

Si n es impar → valor central
Si n es par → (valor[n/2] + valor[n/2+1]) / 2

📊 Ejemplo: Ventas diarias

Sin ordenar: 15, 8, 23, 5, 19

Ordenados: 5, 8, 15, 19, 23

5 8 15 ← mediana 19 23
n (impar)5
Posición central3ra
Mediana15
TEMA 4

Moda

La moda es el valor que más se repite. Puede ser unimodal, bimodal o amodal.

📊 Ejemplo: Tallas de zapatillas vendidas

Datos: 38, 40, 42, 40, 39, 42, 40, 41

38 ×1 40 ×3 41 ×1 42 ×2 39 ×1
Valor más frecuente40
Frecuencia3 veces

TEMA 5

Porcentajes y Percentiles

Los porcentajes expresan cada dato como proporción del total.

% de un dato = (dato / suma total) × 100
Percentil 80 → valor que supera al 80% de los datos

📊 Ejemplo: Ventas por región

Norte: 300 | Sur: 200 | Este: 100 | Oeste: 400 → Total: 1000

Norte30%
Sur20%
Este10%
Oeste40%
TEMA 6

Análisis de Tendencias

Comparar la primera mitad con la segunda mitad de una serie permite saber si los valores suben, bajan o se mantienen.

promedio_primera_mitad vs promedio_segunda_mitad
→ si 2da > 1ra: tendencia ASCENDENTE
→ si 2da < 1ra: tendencia DESCENDENTE
→ si son ≈ iguales: tendencia ESTABLE
